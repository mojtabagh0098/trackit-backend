## Setup

```
git clone https://github.com/Mojtabagh0098/trackit-backend.git
cd trackit-backend
npm install
npm start
```
Finaly go to [TrackIt frontend Repository](https://github.com/Mojtabagh0098/trackit-frontend) and install it
